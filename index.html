Infograph-D3

Skill level infograph Skill level graph to show the expertise level of your skiills on your CV /profile. Created using D3 library.All you need to do is provide data in JSON format.

Default skills infograph

infoGraph.skillLevel.draw("skills",skillsData);

Set your own colours and bubble size

infoGraph.skillLevel.labelColor("#01579B")
		    .bubleColor("#E91E63")
		    .bubleStrokeColor("#03A9F4")
		    .bubleRadius(8)
		    .labelStart(100)
		    .draw("skills",skillsData)

Set your own max level for skill

infoGraph.skillLevel.maxLevel(15)
		    .draw("skills",skillsData)

Don't show max level

infoGraph.skillLevel.showMax(false)
		    .draw("skills",skillsData)

Basic HTML Code

<div id="id you will provide to draw"></div>
